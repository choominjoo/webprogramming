# 박스 모델
## 구성
- 콘텐츠
- 패딩(padding)
- 테두리(border)
- 여백(margin)
## 테두리 스타일
- solid
- none
- hidden
- dotted
- dashed
- double
- groove
- ridge
- inset
- outset
## 모서리 둥근 테두리
- border-radius
## 이미지 테두리 만들기
- border-image
  - round : 에지 이미지 반복 배치. 테두리 길이만큼
  - repeat : 에지 이미지 반복 배치
  - stretch : 에지 이미지 테두리 길이만큼 늘여 배치
## 박스 그림자
- box-shadow
