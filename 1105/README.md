# 1차 회의(1105)

**각자 선정한 주제**

김지우 :  자료 통합 저장소

추민주 : 대학생 시간표 및 시험 일정, 과제 마감일 등 정리하는 사이트 (커뮤니티 X)

이승민 : 랜덤하게 조언 글귀 나오는 사이트

**발표 선정** : 자료 통합 저장소

**선정 이유** : 전체적으로 의논할만한 수정사항이 있고, 배포하였을 때 당장 우리 뿐만 아니라 많은 다양한 사용자들이 찾을 것 같아서. (ex. 학생 뿐만 아니라 교수님들도 찾을 수 있음)

**사용할 툴**

노션, 카카오톡, 디스코드

**수정할 점(1차)**

전체적인 글 검토하고 수정할점 수정하기

주제나 키워드로 검색하는 기능 구현

주제별 자료에서 각각의 카테고리 좀 더 세분화

자료 유형별 필터에 링크들 검토 및 수정

이미지 기능 API 부분 설정해보기

내 과제함에서 메인 화면으로 가는 경로 만들기

위의 기능 구현한 후 내 과제함 사용해보기 (안되면 그때 수정)

**중요한 점**

이런 목적으로, 이런 프롬프트를 줘서, 이런 아웃풋이 나왔다. 라는 결론이 나와야됨. 발표할 때 필요할듯함.
